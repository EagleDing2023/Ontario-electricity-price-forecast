# Ontario Electricity Price Forecast

A machine learning project to forecast Ontario hourly electricity spot prices 24 hours ahead using historical IESO market data.

## Data Sources
- [IESO Hourly Ontario Energy Price (HOEP)](https://www.ieso.ca) — hourly electricity prices 2020-2025
- [IESO Ontario Demand Reports](https://www.ieso.ca) — hourly electricity demand 2020-2025
